# real-estate-website
A modern, responsive real estate company website for showcasing properties, services, and client enquiries

This repository contains a frontend prototype for **RAB Homes Ng Ltd**. The React + Vite frontend is in the `frontend/` folder and includes:

- Premium homepage with hero, featured properties, and CTAs
- Properties listing and detail pages (sample data)
- Team, contact form, and footer

See `frontend/README.md` for usage and deployment instructions.
